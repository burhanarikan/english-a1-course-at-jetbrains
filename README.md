# Basic English for Beginners — A1

> A JetBrains Academy Plugin course for complete English beginners.

## About This Course

This course introduces complete beginners to the fundamentals of the English language at the **A1 level**. Theory, quizzes, and coding exercises are delivered directly inside your IDE via the **JetBrains Academy Plugin**.

Students practice English by writing short Python programs — learning a new language while getting comfortable with a professional development environment.

## Target Audience

- Age 16+
- No prior English knowledge required
- No prior programming knowledge required

## Course Structure (Lesson 1: Basics)

| # | Task | Type | Topic |
|---|------|------|-------|
| 1 | The English Alphabet | Theory | 26 letters, vowels vs consonants |
| 2 | Greetings & Introductions | Theory | Hello, Goodbye, self-introduction |
| 3 | Numbers 0–100 | Theory | Cardinal numbers, spelling rules |
| 4 | Identify the Vowel | Quiz | Single choice |
| 5 | Choose the Correct Greeting | Quiz | Single choice |
| 6 | Number Spelling | Quiz | Multiple choice |
| 7 | Introduce Yourself | Coding | f-strings, print() |
| 8 | Number Dictionary | Coding | dict, for loop |
| 9 | Greetings by Time of Day | Coding | if/elif/else, functions |

## IDE Customization

- **Code style:** PEP 8 (4-space indent, 79-char line limit, 2 blank lines after imports)
- **Inspections:** Spellchecker enabled, type hints & docstrings disabled for beginners

## How to Open

1. Install [JetBrains Academy Plugin](https://plugins.jetbrains.com/plugin/10081-jetbrains-academy) in PyCharm
2. Open this folder as a course project
3. Start learning!
