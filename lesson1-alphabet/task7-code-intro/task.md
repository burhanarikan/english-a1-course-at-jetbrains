# Coding Task: Introduce Yourself

Write a Python program that prints a **self-introduction** in English.

## Expected Output

```
My name is Alice.
I am 20 years old.
I am from Turkey.
```

## Instructions

The variables `name`, `age`, and `country` are already defined.
Use them to print the three lines in the correct format.

💡 **Hint:** Use f-strings!
```python
print(f'My name is {name}.')
```

## Check Your English!

- Line 1: `My name is [name].`
- Line 2: `I am [age] years old.`
- Line 3: `I am from [country].`
